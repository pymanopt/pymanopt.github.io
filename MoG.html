Riemannian Optimisation using Pymanopt for Inference in MoG models¶

The Mixture of Gaussians (MoG) model assumes that datapoints \(\mathbf{x}_i\in\mathbb{R}^d\) follow a distribution described by the following probability density function:

\(p(\mathbf{x}) = \sum_{m=1}^M \pi_m p_\mathcal{N}(\mathbf{x};\mathbf{\mu}_m,\mathbf{\Sigma}_m)\) where \(\pi_m\) is the probability that the data point belongs to the \(m^\text{th}\) mixture component and \(p_\mathcal{N}(\mathbf{x};\mathbf{\mu}_m,\mathbf{\Sigma}_m)\) is the probability density function of a multivariate Gaussian distribution with mean \(\mathbf{\mu}_m \in \mathbb{R}^d\) and psd covariance matrix \(\mathbf{\Sigma}_m \in \{\mathbf{M}\in\mathbb{R}^{d\times d}: \mathbf{M}\succeq 0\}\).

As an example consider the mixture of two Gaussians with means \(\mathbf{\mu}_1 = \begin{bmatrix} -2 \\ 1 \end{bmatrix}\) and \(\mathbf{\mu}_2 = \begin{bmatrix} 0 \\ 0 \end{bmatrix}\), covariances \(\mathbf{\Sigma}_1 = \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix}\) and \(\mathbf{\Sigma}_2 = \begin{bmatrix} 1 & 1 \\ 1 & 3 \end{bmatrix}\), and mixture probabilities \(\pi_1=\frac{2}{3}\) and \(\pi_2=\frac{1}{3}\). Let's generate \(N=1000\) samples of that MoG model and scatter plot the samples:

Given a data sample the de facto standard method to infer the parameters is the expectation maximisation (EM) algorithm that, in alternating so-called E and M steps, maximises the log-likelihood of the data. In arXiv:1506.07677 Hosseini and Sra propose Riemannian optimisation as a powerful counterpart to EM. Importantly, they introduce a reparameterisation that leaves local optima of the log-likelihood unchanged while resulting in a geodesically convex optimisation problem over a product manifold \(\prod_{m=1}^M\mathcal{PD}^{(d+1)\times(d+1)}\) of manifolds of \((d+1)\times(d+1)\) positive definite matrices. The proposed method is on par with EM and shows less variability in running times.

The reparameterised optimisation problem for augmented data points \(\mathbf{y}_i=[\mathbf{x}_i\ 1]\) can be stated as follows:

\[\min_{(S_1, ..., S_m, \nu_1, ..., \nu_{m-1}) \in \prod_{m=1}^M \mathcal{PD}^{(d+1)\times(d+1)}\times\mathbb{R}^{M-1}} -\sum_{n=1}^N\log\left( \sum_{m=1}^M \frac{\exp(\nu_m)}{\sum_{k=1}^M\exp(\nu_k)} q_\mathcal{N}(\mathbf{y}_n;\mathbf{S}_m) \right)\]

where

  • \(\mathcal{PD}^{(d+1)\times(d+1)}\) is the manifold of positive definite \((d+1)\times(d+1)\) matrices
  • \(\mathcal{\nu}_m = \log\left(\frac{\alpha_m}{\alpha_M}\right), \ m=1, ..., M-1\) and \(\nu_M=0\)
  • \(q_\mathcal{N}(\mathbf{y}_n;\mathbf{S}_m) = 2\pi\exp\left(\frac{1}{2}\right) |\operatorname{det}(\mathbf{S}_m)|^{-\frac{1}{2}}(2\pi)^{-\frac{d+1}{2}} \exp\left(-\frac{1}{2}\mathbf{y}_i^\top\mathbf{S}_m^{-1}\mathbf{y}_i\right)\)

Optimisation problems like this can easily be solved using Pymanopt even without the need to differentiate the cost function manually!

So let's infer the parameters of our toy example by Riemannian optimisation using Pymanopt:

from pymanopt.manifolds import Product, Euclidean, PositiveDefinite
from pymanopt import Problem
from pymanopt.solvers import ConjugateGradient
    
import theano
import theano.tensor as T
import theano.compile.sharedvalue as TS
import theano.sandbox.linalg as Tlina

# (1) Instantiate the manifold
manifold = Product([PositiveDefinite(3), PositiveDefinite(3),
                    Euclidean(1,1)])

# (2) Define the cost function (here using theano)
S1 = T.matrix()
S2 = T.matrix()
nu1 = T.matrix()
nu2 = 0
# augment data samples
Y = TS.shared(np.concatenate([samples, np.ones((N,1))], axis=1))
# q_N as defined above
def q_N(y, S):
    return (2*np.pi*np.exp(.5)*
            (T.abs_(Tlina.det(S))**(-1/2))*((2*np.pi)**(-(2+1)/2))*
            T.exp(-.5*y.T.dot((Tlina.matrix_inverse(S)).dot(y))))
# log-likelihood term for one sample
def sampleloglik(y, nu1, S1, S2):
    nusum = T.exp(nu1)+1
    return T.log((T.exp(nu1)/nusum*q_N(y, S1) + 1/nusum*q_N(y, S2)))[0, 0]
# log-likelihoods of all samples
results, updates = theano.scan(fn=lambda y, nu1, S1, S2: sampleloglik(y.T, nu1, S1, S2),
                                  outputs_info=None,
                                  sequences=Y,
                                  non_sequences=[nu1, S1, S2])
# sum of log-likelihoods of all samples
cost = -T.sum(results)

problem = Problem(manifold=manifold, cost=cost, arg=[S1, S2, nu1], verbosity=1)

# (3) Instantiate a Pymanopt solver
solver = ConjugateGradient()

# let Pymanopt do the rest
Xopt = solver.solve(problem)

mu1hat = Xopt[0][0:2,2:3]
